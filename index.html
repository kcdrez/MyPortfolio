Portfolio

Sentinel Comics RPG GM Management Tool
I created a helper tool for Sentinel Comics RPG. The game is a super hero themed role playing game, where one player (the GM) controls the environment and super villains, and other players control super heroes in their adventures. I created the application for my personal games, and have received feedback and feature requests from the Sentinels community.

I used Vue.js, Vuex, and Bootstrap to create reusable components and data management. The GM can add minions and villains, add locations and challenges, and track the movement of the players using a built in virtual table using Fabric.js.
Borrower Wallet
I collaborated with a team of other front end engineers, back end engineers, data analysts, and UI designers to create and update a mortgage fulfilment service for about half a dozen bank clients. Users could fill in the form to apply for a loan, while loan officers could view and submit the loan application.

I used Angular and Angular Material for development, and utilized CI/CD for deployment. See my Git Profile here.
Dire Wolf Digital Website
I updated the company's website to be mobile responsive and have better accessibility on smaller devices. A majority of the website uses PHP and jQuery, as well as some parts using Vue.js for reactivity with user input.
Dire Wolf Digital Online Store
I collaborated on a team to build the company's online store. We used the Braintree API for e-commerce, as well as creating our own mobile responsive design and flow for customers to purchase our products using Vue.js. We created an internal tool as well so that the marketing department can easily add new products or change prices, images, or quantities for existing products, all of which would update the website dynamically.
Pokémon Go Companion Web App
The app uses axios to access an API and displays the data using Vue.js and Bootstrap. Users can search for Pokémon to view type advantages and disadvantages as well as find the highest CP (Combat Power) by each type. I created this app for myself and for friends to aid in our play of Pokémon Go.
Recipe Document
I created my own custom recipe sheet to keep track of all my favorite recipes from various recipe books, family recipes, online recipes, and other sources. The sheets include sections for the following:
