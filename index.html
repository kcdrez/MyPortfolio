About me

I am a Web and Tools Engineer, specializing in creating reactive mobile-friendly web applications using Vue.js and Bootstrap libraries. I have collaborated on numerous projects, both for intranet use as well as public facing content.

I have extensive knowledge with JavaScript, HTML, and CSS and many related technologies. I learn very quickly and have picked up Vue.js within just a few days and have taught myself many other programming languages such as Scala and VBA as my work requests demands. I also have some preliminary experience using C# and C++.

My primary Gallup Strengths are Analytical, Connectedness, Consistency, Empathy, and Intellection. My ability to analyze data quickly and easily is imperative as a developer to be able to find the optimum solution for both consumers, stakeholders, and other developers. I am a consistent in my communication and programming styles which is key for code maintainability and providing documentation for consumers. I believe that having clear goals and transparent information is the best way to build trust within a team and therefore creating the best projects.

I enjoy studying and playings games, both tabletop and digital. Some of my favorites include Battlestar: Galactica the Board Game, Tzolk'in, and Sentinels of the Multiverse. I have even created a few prototype board games of my own!
